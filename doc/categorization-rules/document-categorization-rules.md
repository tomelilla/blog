# 文件分類與排序規則 (Document Categorization and Sorting Rules) - 僅限此專案

為確保此專案的文件管理一致性，請遵循以下規則：

1.  **頂層分類 (Top-Level Categorization)**: 依文件發布年份與月份進行分類，標題格式為 `YYYY年MM月` (例如: `2025年12月`)。
2.  **次層分類 (Second-Level Categorization)**: 在每個年月份分類下，依文件類型細分為 `Markdown 筆記` 和 `PDF 文件`。
3.  **排序規則 (Sorting Rules)**:
    *   在每個次層分類中，文件應依其發布日期進行倒序排列 (最新者在前)。
    *   **文件命名慣例**: 為方便自動化處理，所有文件檔案名請包含發布日期，格式為 `YYYY-MM-DD-文件標題.md` 或 `YYYY-MM-DD-文件標題.pdf`。
4.  **顯示格式 (Display Format)**: 每個文件項目應包含發布日期與文件標題，格式為 `- YYYY/MM/DD - [文件標題](文件路徑)`。
5.  **檔案儲存位置 (File Storage Location)**: 相關文件應儲存於對應年月份的資料夾中，例如 `YYYY-MM/` (例如: `2025-12/`)。
